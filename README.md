# Real-Time Collaborative Code Editor

## Features
- Real-time code collaboration (Socket.io)
- Room-based system
- Code execution with input
- AI error explanation
- Multi-language support
- Dark/Light theme

## Tech Stack
- React
- Node.js
- Express
- Socket.io
- Monaco Editor

## Use Case
- Pair programming
- Interview practice
- Teaching/mentoring

## Limitations
- Uses last-write-wins (no CRDT)

## Future Scope
- CRDT-based collaboration
- Multi-file support
